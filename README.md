YouTube Topmost
===============

Watch YouTube videos into a topmost, undecorated OS native window.

# Demo

![](http://i.imgur.com/fwL6Rn7.png)

After pressing the <kbd>Start</kbd> button, the window will be undecorated and video will be played.
